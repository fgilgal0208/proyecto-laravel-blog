# 🚀 Blog Engine - Proyecto de Prácticas Profesionales

¡Bienvenido! Este repositorio contiene el proyecto desarrollado durante mi periodo de prácticas profesionales en [Taller Empresarial 2.0](https://tallerempresarial.es/). Se trata de un sistema de gestión de contenidos (CMS) tipo Blog, diseñado para ser escalable, moderno y eficiente.

---

## 📋 Descripción del Proyecto
El objetivo principal fue crear una plataforma de blogging completa que permitiera a los usuarios crear, editar y gestionar publicaciones. Durante el desarrollo, me enfoqué en aplicar buenas prácticas de codificación y una arquitectura limpia utilizando el ecosistema de Laravel.

Este proyecto representa mi transición del entorno académico al profesional, enfrentando retos reales de diseño de interfaces y gestión de datos.

## 🛠️ Stack Tecnológico
Para la construcción de este proyecto se utilizaron las siguientes tecnologías:

* **Backend:** [Laravel 10+](https://laravel.com/) (PHP)
* **Frontend:** HTML5, JavaScript (ES6+)
* **Estilos:** [Tailwind CSS](https://tailwindcss.com/)
* **Arquitectura Frontend:** Patrón **Flux** para la gestión del estado.
* **Base de Datos:** MySQL / PostgreSQL.
* **Herramientas:** Vite, Blade Templates, Git.

## ✨ Características Principales
* **Sistema de Autenticación:** Registro e inicio de sesión seguro.
* **Gestión de Posts (CRUD):** Creación, edición y eliminación de artículos con soporte para imágenes.
* **Arquitectura Flux:** Implementada para asegurar un flujo de datos unidireccional y predecible en la interfaz.
* **Diseño Responsive:** Interfaz moderna y adaptable a cualquier dispositivo móvil o de escritorio gracias a Tailwind.
* **Categorización:** Sistema de etiquetas y categorías para organizar el contenido del blog.
